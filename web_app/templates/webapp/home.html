Python Project 1
UDP Group Chat without threads

UDP group chat without threads. Wrote a script “gchat.py” such that when it is called with an argument, which must be an IP address of a server, it acts as a client, and when it is called without an argument, it acts as a server. Both, the client and the server, display every UDP packet coming from the network as a text with a suffix indicating the source, such as “Hello there! by ('154.38.181.152', 12345)”. Whatever text the server receives from keyboard, it transmits the text to all its clients. Whenever the server receives a message from a network, it retransmits the message to all other clients (except to the client where from the message is received). Whenever someone sends a message to the server, the server considers it as a new client.
